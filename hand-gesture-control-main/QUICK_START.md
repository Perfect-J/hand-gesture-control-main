## Switch Modes

Press number keys to switch modes:

- **1** - Media Control (music/video)
- **2** - Mouse Control (cursor)
- **3** - Window Management (organize windows)
- **4** - Presentation Mode (slides)

---

## Need Help?

- Press **H** in the app for help
- Press **C** to pause/resume control
- Press **ESC** to quit

## Practice Routine (5 minutes)

### Minute 1-2: Basic Gestures
- Open hand (spread all fingers)
- Closed fist (curl all fingers)
- Hold each for 2 seconds

### Minute 3-4: Dynamic Gestures
- Swipe left (move hand left quickly)
- Swipe right (move hand right quickly)
- Rotate hand (turn wrist left/right)

### Minute 5: Try Different Modes
- Press **1** for Media
- Press **2** for Mouse
- Press **3** for Windows
- Press **4** for Presentations
